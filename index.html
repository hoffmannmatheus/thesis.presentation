Projeto e Desenvolvimento do eaZy
Visão Geral

Sistema para fornecer informações e controle do ambiente domiciliar.

  • Facilidade de Configuração.
  • Flexibilidade.
  • Criação de Cenários.
Arquitetura
Comunicação por Barramento
  • Comunicação entre os componentes.
  • Linguagens diferentes.
  • Mais de uma implementação por componente.
Home Stack
  • Lógica de Automação Residencial.
  • Gerenciamento de Cenários.
  • Persistência de dados.
Device Controller
  • Interface entre o sistema e os dispositivos reais.
  • Transmite comandos.
  • Recebe atualizações.
eaZy Web
  • Interface de usuário.
  • Interação com dispositivos.
  • Definir cenários.
  • Alertas.
